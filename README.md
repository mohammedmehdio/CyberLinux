## ğŸ“š Commands Included

### Navigation Commands
- `ls` - List directory contents
- `cd` - Change directory

- `cat` - Display file contents
- `less` - Page through files
- `tail` - View end of files
- `head` - View beginning of files

### Network Commands
- `nmap` - Network scanner
- `netstat` - Network statistics
- `ifconfig` / `ip` - Network interface configuration
- `ping` - Test connectivity
- `tcpdump` - Packet analyzer
- `ss` - Socket statistics
- `curl` - HTTP client
- `wget` - File downloader
- `traceroute` - Trace network path
- `dig` / `nslookup` - DNS lookup

### Security Commands
- `sudo` - Execute with elevated privileges
- `iptables` / `ufw` - Firewall configuration
- `ssh` - Secure shell
- `openssl` - Cryptography toolkit
- `gpg` - Encryption and signing
- `fail2ban-client` - Intrusion prevention

### Permission Commands
- `chmod` - Change file permissions
- `chown` - Change file ownership
- `su` - Switch user
- `whoami` - Display current user
- `id` - Display user IDs
- `groups` - Display group membership

### Monitoring Commands
- `ps` - Process status
- `top` / `htop` - System monitor
- `df` - Disk free space
- `du` - Disk usage
- `free` - Memory usage
- `uptime` - System uptime
- `vmstat` - Virtual memory statistics
- `iostat` - I/O statistics

### Forensics Commands
- `grep` - Pattern search
- `strings` - Extract text from binaries
- `file` - Identify file types
- `md5sum` / `sha256sum` - Hash checksums
- `lsof` - List open files
- `strace` - Trace system calls
- `last` - Login history
- `history` - Command history
- `w` - Who is logged in
